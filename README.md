
# -_16# Система управления мероприятиями (Event Organizer)

## Описание выполненной работы

Разработано клиентское приложение на WPF для автоматизации работы организатора мероприятий.

В рамках практической работы были реализованы следующие функции:
1. **Каталог мероприятий (CatalogPage):**
   - Создана страница отображения списка сущностей `Events` в виде визуальных карточек.
   - Использован элемент `ListView` с нестандартной панелью `WrapPanel` для формирования адаптивной сетки.
   - Настроено отображение изображений из BLOB-полей базы данных (с конвертером).
   - Реализована цветовая индикация статусов мероприятий (Planning, Active, Completed, Cancelled).

2. **Работа с данными:**
   - Настроено подключение к базе данных через Entity Framework.
   - Реализована навигация между страницами (Авторизация -> Регистрация -> Каталог).
   - Выполнено наполнение базы реалистичными данными (100+ записей).

3. **Техническая часть:**
   - Использование стилей и шаблонов данных (DataTemplates) в XAML.
   - Конвертеры значений (Value Converters) для обработки изображений и статусов.
